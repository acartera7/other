🧭 Phase 1: Start SIMPLE (No SQL yet)

Before touching your SQL Server, build everything in memory.

👉 Why?
Because LINQ works on collections first — this is where you actually learn it.

✅ Step 1: Create a Console App

In Visual Studio:

New Project → Console App (.NET)
✅ Step 2: Create Your Models

Start with 2 basic classes:

public class Player
{
    public int Id { get; set; }
    public string Name { get; set; }
}
public class Score
{
    public int PlayerId { get; set; }
    public int Points { get; set; }
    public DateTime Date { get; set; }
}
✅ Step 3: Seed Some Fake Data
var players = new List<Player>
{
    new Player { Id = 1, Name = "Alice" },
    new Player { Id = 2, Name = "Bob" },
    new Player { Id = 3, Name = "Charlie" }
};

var scores = new List<Score>
{
    new Score { PlayerId = 1, Points = 100, Date = DateTime.Now },
    new Score { PlayerId = 2, Points = 150, Date = DateTime.Now },
    new Score { PlayerId = 1, Points = 200, Date = DateTime.Now },
    new Score { PlayerId = 3, Points = 120, Date = DateTime.Now }
};
🧠 Step 4: Your FIRST LINQ Leaderboard

👉 Total score per player:

var leaderboard = scores
    .GroupBy(s => s.PlayerId)
    .Select(g => new
    {
        PlayerId = g.Key,
        TotalScore = g.Sum(x => x.Points)
    })
    .OrderByDescending(x => x.TotalScore);

👉 Join with players:

var result = leaderboard.Join(
    players,
    l => l.PlayerId,
    p => p.Id,
    (l, p) => new
    {
        p.Name,
        l.TotalScore
    });

👉 Print it:

foreach (var player in result)
{
    Console.WriteLine($"{player.Name}: {player.TotalScore}");
}
💥 What you just learned
GroupBy
Sum
OrderByDescending
Join

This is core LINQ mastery territory.

🚀 Phase 2: Add Features (Still No SQL)

Now stretch your LINQ muscles:

🔹 1. Add ranking numbers
var ranked = result
    .OrderByDescending(x => x.TotalScore)
    .Select((x, index) => new
    {
        Rank = index + 1,
        x.Name,
        x.TotalScore
    });
🔹 2. Best single score per player
var bestScores = scores
    .GroupBy(s => s.PlayerId)
    .Select(g => new
    {
        PlayerId = g.Key,
        Best = g.Max(x => x.Points)
    });
🔹 3. Recent scores only (last 7 days)
var recent = scores
    .Where(s => s.Date >= DateTime.Now.AddDays(-7));
🧭 Phase 3: NOW Bring in SQL Server

Once you’re comfortable:

✅ Step 1: Create Tables in SQL Server (SSMS)
CREATE TABLE Players (
    Id INT PRIMARY KEY IDENTITY,
    Name NVARCHAR(50)
);

CREATE TABLE Scores (
    Id INT PRIMARY KEY IDENTITY,
    PlayerId INT,
    Points INT,
    Date DATETIME,
    FOREIGN KEY (PlayerId) REFERENCES Players(Id)
);
✅ Step 2: Connect C# to SQL

You’ll use:

System.Data.SqlClient (or newer Microsoft.Data.SqlClient)
✅ Step 3: Load Data into Lists
Read from SQL
Store in List<Player> and List<Score>

👉 Then reuse your LINQ code unchanged

🔥 Phase 4 (Advanced - Optional but Powerful)

Once you're comfortable:

🔹 Compare:
LINQ in memory vs SQL queries

Example SQL leaderboard:

SELECT p.Name, SUM(s.Points) AS TotalScore
FROM Scores s
JOIN Players p ON s.PlayerId = p.Id
GROUP BY p.Name
ORDER BY TotalScore DESC;
